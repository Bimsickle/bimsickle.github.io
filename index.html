Enigma Machine | Python

Whilst reading the book "The Code Book" by Simon Singh, I had the urge to try to code out my own version of an Enigma Machine, albeit in Python.
The most challenging part of this project was to try and adhere to the "plug and rotator" design of the actual Enigma Machine, and also to reverse the algorithms to correctly deicpher the message. Exceptions were made in that only certain ASCII characters are allowed (they get swapped out if they are not on the list, the "machine" still works).
This is an excellent project to tackle and I highly recommend trying it.

GCD | Python

Lastly, an oldie but a goodie (all-time favourite). I wrote this function for my algorithms unit to calculate the GCD of two given numbers. The simplicity is deceptive in that it draws the eye away from the impressive time complexity (it is fast).

def gcd(a,b):
    '''Calculate the GCD of two given numbers'''
    counter=0
    while a%b>0:
        counter+=1
        a,b = b,a%b

    return max(a,b)

print(gcd(5912844,4301996))
